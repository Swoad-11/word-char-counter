Displays word count, character count, and estimated reading time for blog posts. Lightweight, frontend-only plugin with no backend setup required.

== Description ==

Word & Character Counter is a simple and lightweight WordPress plugin that automatically displays the **word count**, **character count**, and **estimated reading time** for your blog posts. 

✅ No backend settings required  
✅ Works out-of-the-box after activation  
✅ Adds a stylish info box above each single post  
✅ Helps improve readability and user engagement  

This plugin is especially useful for bloggers, writers, and publishers who want to give readers an idea of how long a post will take to read.

== Installation ==

1. Upload the plugin folder `word-character-counter` to the `/wp-content/plugins/` directory, or install it through the WordPress admin (Plugins → Add New).  
2. Activate the plugin through the **Plugins** screen in WordPress.  
3. Visit any single blog post to see the word count, character count, and reading time displayed above the content.  

== Frequently Asked Questions ==

= Does it work on pages? =  
No. By default, the counter only appears on **single blog posts** (not pages or archives).  

= Can I customize the style? =  
Yes. You can edit the CSS in the plugin file or override it with your theme’s stylesheet.  

= How is reading time calculated? =  
Reading time is estimated based on an average reading speed of **200 words per minute**.
